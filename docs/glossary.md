# Glossary

- AA: Account Abstraction (ERC-4337).
- DWH: Dual-Witness Handoff protocol for co-signed custody transfer.
- DID: Decentralized Identifier (W3C).
- TTLA: Time-to-Liability-Attribution.
- PUF: Physical Unclonable Function.
- ZK: Zero-knowledge proof.
- ZKVM: Zero-knowledge virtual machine (SP1 or RISC Zero).
- Merkle root: Hash root of ordered handoff bundle data.
- Freshness score: On-chain quality gauge (0-100) updated by proofs.
