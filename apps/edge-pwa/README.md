# Edge PWA (sender/receiver)

Offline DWH app for custody transfer. Runs on rugged phones at handoff points.

Stack: React PWA, Vite, Web Bluetooth, Web NFC, IndexedDB, Yjs.

## Scope
- NFC seal scans and intactness checks
- ZK proof verification from the logger
- Dual signatures and offline queue
- QR display for shipment ID

## Non-goals
- On-chain indexing and analytics
